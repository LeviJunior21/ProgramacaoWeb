Atividades da disciplina

Desenvolvimento de Frontend

Criação da informação visual de um site ou aplicação, com a qual os usuários interagem diretamente. Ele é desenvolvido usando tecnologias como HTML, CSS e JavaScript. Seu objetivo é criar interfaces atraentes, responsivas e funcionais, garantindo uma boa experiência para o usuário.

Desenvolvimento de Backend

Criação da parte de um sistema que lida com a lógica de negócio, banco de dados e integração com outros serviços, funcionando nos bastidores para suportar o frontend. Ele é desenvolvido usando linguagens como Python, Java, Ruby, entre outras. Seu objetivo é garantir que a aplicação funcione corretamente, gerenciando dados e processos essenciais.

Desenvolvimento de Serviços Web

Criação de software que permitem a comunicação entre diferentes aplicações pela internet, usando protocolos como HTTP. Eles facilitam a troca de dados e a integração entre sistemas, geralmente utilizando formatos como XML ou JSON. Exemplos comuns incluem APIs (Application Programming Interfaces) que fornecem funcionalidades específicas para outras aplicações.

O que os alunos falam sobre esta disciplina

Nome do Aluno - período

Programação para Web I
